💰 ราคาที่พัก (ต่อคืน)
ตั้งราคาที่พักเริ่มต้น ราคาเหล่านี้จะใช้คำนวณยอดรวมเมื่อเพิ่มการจอง สามารถปรับราคาในแต่ละการจองได้ตามต้องการ
☁️ Cloud Sync
ข้อมูลออนไลน์ สำรองอัตโนมัติ เมื่อตั้งค่าเสร็จ ข้อมูลการจองจะถูกเก็บไว้บนคลาวด์ฟรี (Firebase) สามารถเปิดใช้งานได้จากหลายเครื่อง ข้อมูลซิงก์อัตโนมัติแบบเรียลไทม์
วิธีตั้งค่า Firebase (5 นาที ฟรี)
1เปิด Firebase Console
ไปที่ console.firebase.google.com แล้วเข้าสู่ระบบด้วย Google Account
2สร้างโปรเจค
คลิก "Add project" → ตั้งชื่อ เช่น "naradee-resort" → กด Continue → ปิด Google Analytics → Create project
3สร้าง Realtime Database
เมนูซ้าย → Build → Realtime Database → Create Database → เลือก Singapore → Start in test mode → Enable
4คัดลอก URL
หน้า Database จะแสดง URL ที่ขึ้นต้นด้วย https://xxx-default-rtdb.firebaseio.com ให้คัดลอกมาวางในช่องด้านบน
5เสร็จสิ้น
กดปุ่ม "บันทึกและเชื่อมต่อ" ข้อมูลจะถูกซิงก์ออนไลน์ทันที สามารถเปิดแอปนี้จากมือถือ/คอมเครื่องอื่น และใช้ URL เดียวกัน เพื่อดูข้อมูลร่วมกัน
